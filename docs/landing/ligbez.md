# Глоссарий

### Термины расположены в алфавитном порядке.

Цыплята - ЦППК

ЦППК - Центральная Пригородная Пассажирская Компания (АО ЦППК)

D4 - Диаметр Горьковского и Киевского направлении

МЦД - Московские Центральные Диаметры

МЦК - Московское Центральное Кольцо

ДТ - Департамент Транспорта Москвы

РЖД - Российские Железные Дороги

6000-х - Пригородный поезд согласно нумерации поездов

7000-х - Экспресс поезд согласно нумерации поездов

Тройка - транспортная карта (ссылка)

Стрелка - транспортная карта для Подмосковье (Ссылка)

Контра - Кассир-Контролер

Коконы - кассир - контролер особого назначение (проводят рейды по подавлению безбилетного проезда)

ТБшники/ТБ - Транспортная безопасность, частная огранизация для обеспечение охраны

Шашлык - зацеперы, которые сгорели от контатной сети

Зацеперы - человек, который едет без билета и на кузове/хвосте/между сцепками/на головных вагонах поезда
